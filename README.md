## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/ViktorHugodev/space-tourism.git
```

```
# Acesse a pasta da aplicação
$ cd space-tourism

# Instale as dependências
$ yarn

# Rode a aplicação
$ yarn dev

```
